Development Timeline

Dev Phases

Chronological phases of the Tribal Zone build — click any phase to expand.

Phase 1 Foundation & Branding Early June 2026 ▼
  • Logo uploaded — logo.png (dragonfly + TRIBAL ZONE text, hot pink)
  • Color theme defined: #F2157B · #120101 · #162950 · #ED1A43
  • Font stack locked: Poppins (body) + Cormorant Garamond (display)
  • Removed Mea Culpa and Great Vibes from all font imports
  • Replaced hardcoded #1c1c1c with #120101 across all files
  • CSS variables standardized: --gold, --dark, --navy, --red
Phase 2 Lili Origin Reference Analysis June 2026 ▼
  • Fetched and analysed liliorigin.com layout in detail
  • Documented: product-first design, horizontal scroll carousels, white bg
  • Catalogued product card specs: hover image-swap, visible "Add to cart", rating format, MRP strikethrough + % OFF
  • Section order mapped: announce bar → header → hero → categories → product rows → banner → footer
  • Collection page: left sidebar + 3-col grid confirmed
  • Created REFERENCE_MAPPING.md with 44 components
Phase 3 Full Redesign — Index, Collection, Product June 2026 ▼
  • Rebuilt index.html — Lili Origin product-first layout
  • Hero 3-slide carousel with arrows + dot nav
  • Category circles horizontal scroll (8 categories)
  • 5 horizontal product carousels: New Arrivals, Earrings, Necklaces, Bestsellers, Zodiac
  • 3-col editorial collection banners
  • Built product.html — gallery, variants, reviews, related products, sticky mobile CTA
  • Rebuilt all 6 collection pages with sidebar filters + product grid
Phase 4 UX Fixes & Polish June 12, 2026 ▼
  • Added hover-to-zoom magnifier on product gallery (2.4× mouse-tracking)
  • Fixed price row alignment — stacked layout + 2-line min-height on product name
  • Fixed cross-referenced images — earring photos showing on bracelet/maang tikka cards
  • Audited all 14+ product cards, assigned category-correct Unsplash images
  • Identified photo-1611652022419 as necklace photo (misused on earring cards), replaced everywhere
  • Fixed auto-push hook — rewrote from bash+jq to PowerShell autopush.ps1
Phase 5 Remaining Pages Upcoming ▼
  • Build collection-hampers.html — Festive Hampers page
  • Build collection-sale.html — Sale page with red styling
  • Real product photos replacing Unsplash placeholders
  • Shopify Liquid sync — mirror all HTML changes to .liquid files
  • Live store deployment on tribalzone.in
